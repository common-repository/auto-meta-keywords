== Description ==

This plugin automatically gets the keywords of your post/page content and shows them in the meta keywords tag. Meta keywords tag can be used when determining the page relevance to search queries by the search engines.
Example of meta keywords tag would be like this

meta name="keywords" content="blogging, hiking, writing, skills"

Its comptiable with popular page builders like Elementor, Divi, Wpbakery, Gutenberg editor etc. Super fast, doesn't effect page load speed. 
Use the filter "amk_custom_keywords" to set custom keywords for any specific post.
